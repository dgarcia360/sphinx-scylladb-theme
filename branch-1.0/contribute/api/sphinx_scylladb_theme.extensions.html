sphinx_scylladb_theme.extensions.navigation module¶

Extension for Sphinx that generates a navigation tree with dropdowns from Sphinx’s toctree function’s output.

Adapted from https://github.com/pradyunsg/furo (Copyright (c) 2020 Pradyun Gedam, MIT License) for sphinx-scylladb-theme.

sphinx_scylladb_theme.extensions.navigation.get_navigation_tree(toctree_html: str, collapse: bool) → str¶

Modify the given navigation tree, with furo-specific elements. Adds a checkbox + corresponding label to <li>s that contain a <ul> tag, to enable the I-spent-too-much-time-making-this-CSS-only collapsing sidebar tree.

sphinx_scylladb_theme.extensions.navigation.side_nav_has_one_item(toc: str) → bool¶

Check if the toc has exactly one list item.
